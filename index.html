About Me

Hello! My name is Lane and I enjoy creating things that live on the web. My interest in web development starting with updating MySpace and Xanga themes for my friends.

Fast-forward to today, and I've had the privilege of working with data analytics, backend developement, and UI/UX design. My main focus is on building accessible, inclusive products as a full-stack developer.

Full-stack development is one of the most creative areas of IT. I do it for the personal satisfaction of making a big difference in the companies I work with! Being a full stack developer means that I design and code every phase of a program, from the user side to the server. In short, I am a versatile software developer who needs to know both the user, server, and database directions.

Experience

Back End Development

Proficent with the use SQL to allow the end users to interact with the underlying databases. SQL is a great tool to pull business analytics.

Front End Development

Experienced with creating modern, beautifully designed websites with HTML, CSS and C# that feel great to use.

Programming Languages

HTML, CSS, PHP, JavaScript, jQuery, C#, Swift

Shopify Development

Versed in Shopify e-commerce web design and development.

WordPress Development

With the focus of improving the infrastructure and ecosystem of the wordpress platform.

Frameworks and Libraries

Gaining familiarity with .Net core for developing applications and software that ease your business processes.
